# 📊 Sales Analysis Project (Python + Power BI)

## 🚀 Project Overview
This project presents an end-to-end sales analysis workflow using Python and Power BI.  
It includes data cleaning, preprocessing, and an interactive dashboard to generate business insights.

---

## 🔄 Workflow
1. Data loading and exploration using Python  
2. Data cleaning and preprocessing  
3. Feature engineering (Year, Month)  
4. Dashboard creation using Power BI  

---

## 🐍 Python (Data Cleaning & Analysis)
- Handled missing values  
- Converted data types (Date, Sales)  
- Removed inconsistencies in column names  
- Created new features (Year, Month)  
- Exported cleaned dataset  

📂 Files:
- `Python/data_cleaning.ipynb`  
- `Python/sales_data_sample.csv`  
- `Python/clean_sales_data.csv`
